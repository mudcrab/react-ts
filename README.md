# React TS

An opinionated starting point for React projects with Typescript

### Includes

- React
- Typescript
- Redux (with Rematch)
- React Router 5
- React Hook Form
- ky for requests
- Tailwind

### How?

- `git clone https://github.com/mudcrab/react-ts YOUR_PROJECT`
- `cd YOUR_PROJECT ; yarn`
- `yarn dev`

#### Production

- `yarn build`
